W04 Website Project Site Plan

Project Title: Southern California International Student Life Guide

Author: Yang Liao

1. Site Name

Southern California International Student Life Guide

This name clearly reflects the main purpose of the site: a practical guide for international students who are living and studying in Southern California. It is descriptive, easy to understand, and can be used together with a simple logo and favicon later in the final project.

2. Site Purpose

The website will provide essential and practical information for international students who attend community colleges, BYU-Pathway Worldwide, Cal Poly Pomona, and other universities in Southern California. The goal is to help newcomers handle common challenges such as housing, transportation, scholarships, work rules, and school / course planning.

The site will also demonstrate modern web development skills from this course, including semantic HTML, responsive CSS, and JavaScript features such as data fetching, local storage, and interactive components.

3. Target Audience & Scenarios

Target Audience: New and current international students in Southern California (F-1 / J-1, or other visa types) who need simple, experience-based information in English.

Scenarios (User Questions)

  • “Where can I find affordable housing options near community colleges such as Chaffey College or Citrus College?”
  • “How do I use public transportation (bus / metro) in Southern California and are there any student discounts?”
  • “What scholarship and financial resources are available for international students, including BYU-Pathway and local community colleges?”
  • “What are the basic F-1 / J-1 work rules in the U.S., and what should I know before working on campus or during CPT/OPT?”
  • “How can I plan my classes and transfer credits between schools in a smart way?”

4. Planned Pages for the Final Website

Page 1 – index.html (Home – Overview)

  • Short welcome message and introduction to Southern California student life.
  • Hero section with key call-to-action buttons (Housing, Money & Work, School Planning).
  • Quick overview cards that summarize the three main content areas.
  • Link to the HTML form page (for questions / simple contact) and to the Attributions page.

Page 2 – living.html (Living Resources)

  • Information about housing types (homestay, shared apartment, student housing, etc.).
  • Simple tips about renting in California (lease, deposit, roommate, safety).
  • Transportation section with bus / metro basics and student discounts.
  • Dynamic list (from JSON) of example housing or living resources in Southern California.

Page 3 – study-planning.html (Academic, Money & Course Planning)

  • Scholarship and financial aid ideas that are realistic for international students.
  • Short introduction to things like Phi Theta Kappa, BYU-Pathway programs, and transfer planning.
  • Tips for choosing classes, combining community college + online courses, and not falling behind.
  • Dynamic cards (from the same JSON or another JSON section) listing programs / resources with links.

The HTML form and the attributions page will be additional simple pages (form action + resource references) as required in the project description, but they will not count as part of the three main content pages.

5. Color Scheme

Primary Color – Deep Blue #003366

Usage: main headings, header background, navigation bar, footer background.

Accent Color – Bright Cyan #00A3C8

Usage: buttons, links on hover, small badges, focus states, important icons.

Neutral Background – Light Gray #FAFAFA

Usage: main page background to make the content easy to read.

Text – Dark Gray #333333

Usage: body text for good contrast and readability.

6. Typography

Primary Font: Inter (Google Font)

  • Headings (H1–H3): Inter 700, uppercase or sentence case.
  • Body text: Inter 300–500, comfortable line height.
  • Navigation & buttons: Inter 600, slightly larger letter spacing.

Inter is a modern, clean sans-serif font that works well on screens and supports a professional but friendly tone for international student content.

8. Planned JavaScript Features (for Final Project)

To align with the final individual project description, this site will include the following JavaScript features:

  • Data Fetching: Use the Fetch API to load a local JSON file that contains at least 15 resource items (for example: housing resources, scholarship programs, transportation links, and study tools). Use async/await with try...catch for error handling.
  • Dynamic Content: Generate cards on the Living and Study Planning pages from the JSON data. Each card will show at least four properties (name, type, city/area, short description, and a link or tag).
  • Local Storage: Allow users to “save” or “favorite” certain resources (for example, a housing link or scholarship). Store these favorites in localStorage and show them in a small “Saved for Later” section.
  • Modal Dialog: When a user clicks “More Info” on a resource card, open an accessible modal dialog that shows extra details and a link. The modal can be closed with a close button or the Escape key.
  • HTML Form + Action Page: A simple “Ask a Question / Contact” form that collects name, school, email, and a short question. The form will POST to a basic action page which echoes the submitted data.
  • Array Methods & Template Literals: Use map / filter / forEach to process the JSON data and build the HTML with template literals.
  • ES Modules: Split the JavaScript into modules (for example, data-service.js, ui.js, main.js) to keep the code organized.

9. Reason I Chose This Subject

I am an international student living in Southern California and currently studying at multiple schools, including Cal Poly Pomona, BYU-Pathway, and local community colleges. I have personally faced many challenges related to housing, transportation, scholarships, work restrictions, and choosing classes. Because many new students experience similar problems, I want to create a simple, experience-based, and helpful guide that can make their first months in Southern California a little easier.
